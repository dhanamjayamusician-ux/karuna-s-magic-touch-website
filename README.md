# Karuna's MAGIC TOUCH Website

Static website for Karuna's MAGIC TOUCH, a women-owned bridal makeup and beauty parlour in Gopalapatnam.

## Files

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`

## Publish With GitHub Pages

1. Create a new GitHub repository named `karunas-magic-touch-website`
2. Upload all files from this folder
3. Open the repository `Settings`
4. Open `Pages`
5. Under `Build and deployment`, choose `Deploy from a branch`
6. Select branch `main`
7. Select folder `/ (root)`
8. Save

GitHub will generate a live demo link after deployment.
